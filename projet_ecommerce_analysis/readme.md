# E-commerce Data Analysis (BigQuery)

This project explores an e-commerce dataset using SQL and analytical techniques.

## Objectives
- Analyze sales evolution over time
- Build time-series metrics using window functions
- Apply BigQuery performance optimizations
- Present insights through a dashboard

## Tech Stack
- BigQuery (SQL)
- Jupyter Notebook
- Looker Studio

## Project Structure
- Notebook: analytical reasoning and SQL exploration
- Dashboard: visualization of key KPIs

## Notes
The dataset is synthetic and used for learning and demonstration purposes.
